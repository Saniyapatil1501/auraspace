# 🌙 AuraSpace – AI Mood Journal

AuraSpace is a modern, aesthetic mood tracking and journaling web app designed to help users understand their emotions, build mindfulness, and maintain mental well-being through daily reflections.

---

## ✨ Features

* 🧠 Mood tracking with emoji-based selection
* ✍️ Personal journaling system
* 💭 Daily prompts for guided reflection
* 📊 Dashboard with mood insights & streaks
* 💾 Secure data storage using Supabase
* 🎨 Smooth animations using Framer Motion
* 🌗 Dark / Light theme support
* 💖 Clean glassmorphism-based UI

## 🛠 Tech Stack

* **Frontend:** React + TypeScript + Vite
* **Styling:** Tailwind CSS + ShadCN UI
* **Animations:** Framer Motion
* **Backend:** Supabase (Auth + Database)
* **Deployment:** Vercel

---

## 🔐 Authentication

* Email & Password authentication via Supabase
* Persistent login sessions
* Secure user-based data access

---

## 📂 Project Structure

```
src/
 ├── components/      # Reusable UI components
 ├── pages/           # App pages (Dashboard, Mood, Profile, etc.)
 ├── lib/             # Utility functions and configs
 ├── integrations/    # Supabase client setup
 └── hooks/           # Custom React hooks
```

---

## ⚙️ Environment Variables

Create a `.env` file in the root:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_anon_key
```
